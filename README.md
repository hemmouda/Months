# Months &nbsp; ![DEVELOPMENT STATUS: complete](https://badgen.net/badge/DEVELOPMENT%20STATUS/complete/green)

> Does June have 31 days or 30? Or is that July? Wait, which one is the 6th month again?

## Usage examples:

Display info about the 6th month:

```console
$ months 6
```

Display info about all months:

```console
$ months
```

![Example](https://github.com/user-attachments/assets/d72adb71-e985-443e-8944-cdcf4307cf1a)

## How-to:
1. Put the **months** script with your other **bin**aries, or in a separate folder, but then don't forget to include that folder to your **PATH** variable (There are plenty of resources out there if you don't know how to do the latter)
2. Add execution rights to it: ```$ chmod u+x months```
3. Rehash to use instantly: ```$ rehash```
